# Pass the Wheel 🚗

> *Study smart. Pass the wheel.*

A free, no-login study tool for the **Washington State DOL knowledge test** — built on the official WA Driver Guide.

## What's inside

- **65 real exam-style questions** across all 5 tested categories
- **Practice mode** — filter by topic, infinite shuffle, streak tracker
- **Challenge bank** — automatically captures every question you miss and drills it until you clear it
- **Full exam simulator** — 40 questions, timed, same format as the real DOL test
- **2-week study plan** — day-by-day roadmap to test day
- **Exam secrets** — the 10 highest-frequency traps on the real WA DOL exam
- **Progress saves automatically** — picks up where you left off every time you return

## Who it's for

Anyone preparing for the Washington State DOL written knowledge test — new drivers, people transferring a license from another state or country, or anyone who needs a refresher.

## How to use it

Just visit the link below — no download, no account, no app needed:

👉 **[passthewheel — live site](https://eniloops.github.io/pass-the-wheel)**

Or download `index.html` and open it in any browser on your computer. Works 100% offline.


## Built with

Plain HTML, CSS, and JavaScript — zero dependencies, zero frameworks. One file. Opens anywhere.
